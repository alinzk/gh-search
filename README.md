# GitHub repo search

A basic UI implmentation for GitHub repo search

## Dependencies
- Ruby 2.7.1
- Rails 6.0.2
- Nodejs

## To setup project
- Install deps using `bundle install` & `yarn install`
- Run `rails s` at project root, fire up the browser and go to `localhost:3000`

## Specs
- For rails specs `rspec`

## Lint
- `rubocop`
